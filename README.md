# java-financial-utility-tools
A suite of Java utility applications focusing on financial calculations, user input validation, and conditional logic processing.

The Goal: To develop a collection of practical tools, including a mortgage payment calculator with compound interest logic and a simulated banking interface, that handle complex user inputs and numerical precision.

The Challenge: Ensuring data integrity during financial transactions (eg. preventing negative balances in the BankingApp) and implementing strict input constraints in the Mortgage Calculator to prevent calculation errors.

The Solution: I used while loops for robust input validation and NumberFormat classes to ensure concurrency was displayed correctly. I implemented switch-case menus to create a seamless user experience within the console interface.
